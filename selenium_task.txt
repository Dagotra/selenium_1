Дополнительная информация:
Задача - Внедрите автоматизированные тесты на основе предоставленного тестового примера.
Требования:
1. Библиотека для тестирования - PyTest.
2. CVS - Git.
3. Для работы с элементами браузера на страницах следует использовать библиотеку Selenium WebDriver.
4. Тест должен работать в браузере Chrome.
Сайт - https://store.steampowered.com/

Тестовый сценарий:
Перейти на главную страницу - Отображается главная страница.
Нажмите кнопку входа - Страница входа открыта.
Введите случайные строки в качестве учетных данных. Нажмите кнопку «Войти».:
 1. Отображается элемент загрузки.
 2. Отображается текст ошибки (после того, как
элемент загрузки исчезнет).
